# Security Policy

## Supported Versions

Currently, only the latest version of QuickDraw is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do not report it through public issues.

Instead, please report it privately to the project maintainer by sending an email to [INSERT EMAIL].

Please include the following in your report:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact of the vulnerability.

We will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible.
