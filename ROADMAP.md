# QuickDraw Roadmap 🚀

This document outlines the planned features and improvements for QuickDraw.

## Phase 1: Core Experience (Completed ✅)
- [x] Real-time stroke synchronization
- [x] Room-based collaboration
- [x] AI Design Assistant integration
- [x] Basic shape tools (Rectangle, Circle, Line, Arrow)
- [x] Monorepo architecture with Turborepo
- [x] CI/CD Pipeline & Labeler

## Phase 2: Enhanced Creativity (In Progress 🏗️)
- [ ] **Hand-drawn mode**: Toggle between perfect shapes and Excalidraw-style "rough" strokes.
- [ ] **Layers Panel**: Manage drawing layers for complex designs.
- [ ] **Image Uploads**: Drag and drop images directly onto the canvas.
- [ ] **Export Options**: Export to SVG, PNG, and high-quality PDF.

## Phase 3: Team Collaboration (Upcoming 📅)
- [ ] **Voice Chat**: Built-in audio for seamless brainstorming.
- [ ] **Permissions System**: Read-only vs. Editor access for room members.
- [ ] **Recording & Playback**: Record a drawing session and play it back like a video.
- [ ] **Templates Gallery**: Pre-built templates for flowcharts, wireframes, and mind maps.

## Phase 4: Platform & Ecosystem
- [ ] **Offline Mode**: Local storage support with background sync.
- [ ] **Mobile App**: Dedicated iOS and Android versions using React Native.
- [ ] **Plugin System**: Allow developers to build custom tools for the canvas.

---
*Note: This roadmap is subject to change based on community feedback.*
